# `~/Code` structure

My active repositories mostly live under `~/Code/{kind}/{repo}`, where `{kind}` groups related work.

When working from a repository inside a `{kind}` directory and I mention another repository, it is most likely a sibling in that same `{kind}` directory. Look there first. If it isn't there and needs cloning, ask me whether to clone it into `/tmp` or into the `{kind}` directory as a sibling.
